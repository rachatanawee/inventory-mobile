# APK Files

Place your built APK files here.

To build the APK:
1. Go to `inventory-mobile` directory
2. Run `bun run build:android`
3. Copy the APK from `inventory-mobile/android/app/build/outputs/apk/release/app-release.apk`
4. Rename it to `inventory-system.apk` and place it here

The APK will be available at: `http://localhost:3000/apk/inventory-system.apk`
